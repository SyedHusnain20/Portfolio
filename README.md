A Pakistani Legal Advisory Chatbot powered by Retrieval-Augmented Generation. Embeds legal documents using `sentence-transformers`, retrieves relevant passages via FAISS, and generates grounded answers through Groq's Llama3-70b. Served via FastAPI.
* **Tech:** RAG, FAISS, Sentence-Transformers, Groq / Llama3-70b, FastAPI, Jinja2, NLP

An AI-powered resume screening API that parses resumes, extracts skills via spaCy NER, and scores candidates against job descriptions using ML. Fully containerized.
